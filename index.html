Approximation of Rational Intervals by Equal Temperaments

(Inspired by this Mastodon post by Mike Stay. And in the meantime John Carlos Baez has also announced another similar diagram.)

For a given value n and a natural interval (a frequency ratio between two small integers), how closely can the interval be approximated with n-tone equal temperament?

  • The numerator and denominator of the ratio can be selected below.
  • The horizontal axis of the diagram gives the number n of steps per octave.
  • The vertical axis gives the difference between the ratio and the approximation, measured "on the logarithmic scale".
  • Depending on the first checkbox below, signed or unsigned differences are displayed.
  • The other checkbox selects whether the difference is measured in cents (1/1200 of an octave) or in multiples of a step. (In the latter case the difference is always between -0.5 and +0.5.)
  • The dots in the graph sometimes line up to several "strands". Actually the n values of a strand are from a residue class modulo the number of strands.
    The number of strands can be selected below.
    We get "nice" (relatively flat) strands if the last number below (the interval in octaves times the number of strands) is close to an integer.

(Source)

ratio = numerator / denominator interval in octaves = log2(ratio) interval in octaves * #strands
